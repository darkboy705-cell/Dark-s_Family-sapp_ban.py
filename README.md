​🚀 WhatsApp Ultimate Turbo Tool (V2.0)
​Un outil d'automatisation puissant conçu pour le Bombing et l'envoi de Supports Multimédias. Ce dépôt contient deux versions optimisées : une pour PC et une pour Mobile (Termux).
​⚠️ AVERTISSEMENT : Cet outil est créé à des fins éducatives uniquement. L'utilisation abusive peut entraîner le bannissement de votre compte par Meta. L'auteur décline toute responsabilité.
​✨ Fonctionnalités Globales
​💣 Mode Turbo Bombing : Envois massifs ultra-rapides.
​📁 Support Médias : Envoi automatique d'images, vidéos et documents.
​🛡️ Anti-Ban System : Délais aléatoires pour simuler un comportement humain.
​🔄 Session Persistence : Une seule connexion requise, l'outil mémorise l'appareil.
​💻 PARTIE 1 : Utilisation sur PC (Windows / Linux / Mac)
​Idéal pour une gestion visuelle avec Chrome.
​Prérequis
​Python 3.x installé.
​Navigateur Google Chrome.
​Installation & Lancement
Cloner le projet :git clone https://github.com/TonPseudo/ton-depot.git
cd ton-depot
Installer les dépendances :pip install -r requirements.txt
Lancer l'outil :python main.py
📱 PARTIE 2 : Utilisation sur Android (Termux)
​Optimisé pour un usage sur un seul téléphone sans besoin de scanner de QR Code.
​Installation Automatique
Donner les permissions au script :chmod +x install.sh
Lancer l'installateur :./install.sh
Lancement en Mode "Solo Device"
​Pour lier votre compte sans deuxième écran :
Lancer le script Node.js :node termux_bomb.js
Liaison par code : Un code à 8 caractères s'affiche dans Termux.
Action sur WhatsApp : Allez dans Paramètres > Appareils liés > Lier avec le numéro de téléphone et saisissez le code.
